HTML Code Examples

These are simple code examples I practised during my learning.

Example 1 — Basic HTML Page

This is the minimum code needed for a valid HTML5 page. Every HTML file should start with this structure. 

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My Page</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>
