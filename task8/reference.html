HTML Tag Reference

A quick reference table of common HTML tags I use in my projects.
Tag Name What it does
<html> Root element Wraps the entire HTML document
<head> Head section Contains meta info, title, favicon links
<body> Body section Contains all visible page content
<h1> to <h6> Headings Six levels of heading from largest to smallest
<p> Paragraph Wraps a block of text as a paragraph
<a> Anchor / Link Creates a hyperlink to another page or URL
<img> Image Embeds an image. Needs src and alt attributes
<table> Table Creates a table with rows and cells
<form> Form Groups form inputs for user data collection
<input> Input field Accepts user input — text, email, password etc.
<ul> <ol> Lists Unordered (bullet) and ordered (numbered) lists
<section> Section Groups related content into a thematic section

Attribute Quick Reference

Common img attributes

src — path to the image file
alt — description for screen readers
width & height — size in pixels
title — tooltip text on hover
border — adds a border around the image


Common a (link) attributes

href — the destination URL or file path
target="_blank" — opens in a new tab
title — tooltip description for the link
download — triggers file download instead of navigation
rel — relationship between current and linked page


Accessibility attributes

alt — alternative text for images
aria-label — provides a label for screen readers
role — defines the purpose of an element
lang — declares the language of an element
tabindex — controls keyboard tab order
