Getting Started with HTML

This page will help you set up your environment and write your very first HTML file. It is written in simple language so even complete beginners can follow along easily.

Step 1 — Install a Text Editor

Download and install Visual Studio Code from code.visualstudio.com . It is free and works on Windows, Mac, and Linux. You can also use plain Notepad if VS Code feels too heavy.

Step 2 — Create Your First File

Open VS Code, create a new file, and save it as index.html. The .html extension tells the browser it is a web page. Type the basic HTML structure shown below and save the file. 

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

Step 3 — Open in Browser

Right-click your index.html file and choose Open With → Chrome (or any browser). You should see your heading and paragraph on screen. That is it. You just created your first web page!

Common Beginner Mistakes

Forgetting to close tags

Every opening tag like <p> needs a closing tag </p>. If you forget the closing tag, the browser may render the rest of your page incorrectly. Always double-check your tags are properly closed.


Wrong file extension

Save your file with .html not .txt. If you save it as a text file the browser will show the raw code instead of rendering it as a web page. In Windows make sure hidden file extensions are visible.


GitHub Setup Basics

To push your HTML project to GitHub:

  1. Create a free account at github.com
  2. Install Git from git-scm.com
  3. Run git init in your project folder
  4. Run git add . then git commit -m "first commit"
  5. Push using git push origin main
