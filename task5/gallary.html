# Feature Description Code Example Used
1 SVG Emoji Favicon A chain-link emoji is set as the browser tab icon using an inline SVG data URI. <link rel="icon" href="data:image/svg+xml,...">
2 Table-Based Page Layout The entire page (header, nav, sections, footer) is structured using HTML tables with width, bgcolor, cellpadding, and cellspacing. <table width="100%" bgcolor="#2c3e50">
3 Navigation Bar with Links A horizontal nav bar with anchor links to other pages (Home, Tables, Form, Navigation, Contact). <a href="../task1/index.html">Home</a>
4 figure & figcaption Images are wrapped in semantic <figure> elements with <figcaption> labels below them. <figure><img ...><figcaption>Title</figcaption></figure>
5 Image Attributes Every image uses alt, width, height, border, hspace, vspace, and title for accessibility and spacing. <img alt="..." width="220" height="155" border="1" hspace="5" vspace="5" title="...">
6 Linked Image — target="_blank" An image is wrapped in an anchor tag that opens the MDN HTML Docs in a new browser tab. <a href="https://..." target="_blank"><img ...></a>
7 Linked Image — target="_self" Another image links to GitHub and opens in the same tab using target="_self". <a href="https://github.com" target="_self"><img ...></a>
8 Relative Image Paths (3 types) Images use same-folder paths (./images/), sibling-folder paths (../task5/images/), and parent-folder paths (../assets/). src="./images/vscode.jpg"
src="../task5/images/..."
src="../assets/..."
9 Absolute URL Image One image is loaded directly from the internet using a full Unsplash URL, demonstrating external image embedding. src="https://images.unsplash.com/photo-...?w=600"
10 Image Text Wrap (align="left" / "right") Two images use align="left" and align="right" to wrap descriptive paragraph text around them, with br clear="all" to reset. <img align="left"> ... text ... <br clear="all">
11 <picture> Responsive Element A <picture> element with two <source> media queries serves different images at different screen widths, with a fallback <img>. <picture>
<source media="(min-width:800px)" srcset="...">
<source media="(min-width:400px)" srcset="...">
<img src="fallback.jpg">
</picture>
12 Coloured Section Headers Each gallery section has its own colour-coded header row (dark blue, green, purple, orange) to visually separate topics. <tr bgcolor="#27ae60"><h2>...</h2></tr>
13 Font Styling (face, size, color) The <font> element is used throughout for Arial typeface, specific sizes, and hex colour values on all text. <font face="Arial" size="2" color="#555555">...</font>
