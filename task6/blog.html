My HTML Learning Journey

Posted on January 10, 2024  —  by Paladugu Venkata Bharath

I started learning HTML in my second year of BE CSE. At first I thought it would be very hard but once I started writing basic tags like <h1> and <p> it felt easy and interesting. HTML is the foundation of every website and I am glad I learned it properly.

The first webpage I made was a simple page with my name, college details, and a list of my skills. It had no design but it worked in the browser and that made me very happy. I used VS Code as my code editor and the Live Server extension to preview changes in real time. That feeling of seeing your code work in a browser for the first time is really amazing.

Things I Learned in HTML

I learned how to use headings, paragraphs, lists, tables, forms, and links in HTML. I also learned about semantic tags like <header>, <nav>, <main>, <section>, and <footer>. These tags help make the page structure clear and easier to understand for both humans and search engines.

A Note on HTML Versions

We are currently using HTML5 which is the latest version. Older versions like HTML 4 and XHTML are no longer used for new projects. HTML5 added many useful new elements like <video>, <audio>, and <canvas>. The water molecule is H2O and area is measured in m2 — just two examples of sub and sup tags in real use.

"HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page and consists of a series of elements."
— MDN Web Docs

My college professor always said learn HTML before jumping to frameworks and I think that advice was very useful. It helped me understand how websites actually work from the ground up.

My First HTML Code 

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

Learning GitHub for the First Time

Posted on February 5, 2024

When I first heard about Git and GitHub I had no idea what they were. A senior student explained that GitHub is used to store and share code online and also helps teams work on the same project together. That explanation made me want to learn it immediately.

Basic Git Commands I Use Daily

To start using Git I learned a few basic commands. I type git init to start a new repository, then git add . to add all files, and git commit -m "first commit" to save changes. After that git push origin main uploads the code to GitHub. When the push works the terminal shows:

Branch 'main' set up to track remote branch 'main'.
Everything up-to-date.

"Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency."
— Git Official Website

One of my batchmates told me once you start using GitHub you cannot go back to just saving files on your desktop and honestly that is very true. I now push all my projects to GitHub even small ones.

Variable Example in Git Workflow

In shell scripts related to git automation, we often use variables like branch_name and commit_message to make reusable deployment scripts. This is a beginner-level concept covered in DevOps basics.


Getting Started with Python

Posted on March 12, 2024

Python is the second programming language I learned after HTML. It is much easier than I expected. Python uses simple English-like syntax which makes it great for beginners. I started with printing hello world, then moved on to variables, loops, and functions. Python made me enjoy programming more.

A Simple Python Program 

# Simple Python program
name = "Bharath"
age = 21

print("Hello, my name is", name)
print("I am", age, "years old")

for i in range(5):
    print("Learning Python - Day", i+1)

The variable name stores a string value and age stores an integer. I used to think Python was only for data science but now I know it is used for web development, automation, and scripting too. It is one of the most popular languages in the world right now.

"Python is a programming language that lets you work quickly and integrate systems more effectively."
— Python.org

My trainer at the internship told us Python is not just a language, it is a superpower for developers. I totally agree after seeing how much you can do with just a few lines of Python code.

Note: Python version used is 3.11

All examples on this blog use Python 3.11. Python 2 is no longer supported since January 2020. Please make sure you install the correct version from python.org before following along.

Tech Terms I Learned

HTML
HTML is the standard language used to create web pages. It uses tags to structure content in a browser.

GitHub
GitHub is a website where developers store and share code using Git version control. It is widely used in real software projects.

Python
Python is a beginner-friendly programming language known for its clean and simple syntax. It is used in web development, automation, and data science.

Browser
A browser is a software application used to access websites. Examples include Chrome, Firefox, and Edge. It reads HTML files and displays them as web pages.

Database
A database is an organised collection of data. I use SQL with MySQL to create tables, insert records, and run queries during my college lab sessions.
