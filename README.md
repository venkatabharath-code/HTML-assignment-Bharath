## Task 2 — Advanced Link Implementation & Navigation Patterns

In this task, I developed a comprehensive `navigation.html` webpage to demonstrate advanced HTML linking and navigation techniques using PURE HTML only, without JavaScript, external CSS, or frameworks. The page was designed with a structured layout using semantic HTML elements such as `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>` along with inline styling attributes for presentation.

The project includes a fully functional Table of Contents at the top of the page using an ordered list with internal anchor links (`#id`) that navigate users to different sections within the same page. Each major section also contains a “Back to Top” link to improve navigation and user experience across long-form content.

Relative linking techniques were implemented by connecting multiple HTML pages organized in a proper folder structure:
- `pages/about.html`
- `pages/contact.html`
- `resources/help.html`

The navigation system was carefully configured using correct relative paths to ensure smooth movement between all project pages.

Download functionality was added using the HTML `download` attribute, allowing users to download PDF and TXT files directly from the webpage. Email and telephone hyperlinks were implemented using `mailto:` and `tel:` protocols with proper formatting, including predefined email subject and body content.

An interactive image map was also created using `<map>` and `<area>` elements with multiple clickable regions using different shapes such as:
- Rectangle (`rect`)
- Circle (`circle`)
- Polygon (`poly`)

Each clickable region redirects users to specific sections within the webpage, demonstrating advanced image navigation concepts.

The Contact page was additionally enhanced with a professional HTML form containing:
- Text input
- Email input
- Telephone input
- Subject dropdown menu
- Textarea message box
- Submit and reset buttons
- Proper `<label>` tags
- HTML validation attributes

Throughout the task, semantic HTML structure, accessibility practices, descriptive text, organized folder hierarchy, favicon integration, inline styling, and clean code formatting were maintained according to the assessment guidelines.

## Task 4 — Comprehensive Form Development with Validation

In Task 4, I developed a professional and fully structured HTML registration form using PURE HTML only, following all assessment guidelines and validation requirements. The form demonstrates advanced usage of HTML form elements, semantic structure, accessibility practices, and built-in validation features without using JavaScript, external CSS, or frameworks.
